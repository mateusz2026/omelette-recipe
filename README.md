# Omelette Recipe Page

## 📌 Overview
This project is a solution to the Frontend Mentor Omelette Recipe challenge.  
The goal was to build a clean and responsive recipe page layout.

## 🚀 Live Demo
https://mateusz2026.github.io/omelette-recipe/

## 🛠️ Built with
- HTML5
- CSS3
- Flexbox

## 📱 Features
- Clean and readable layout
- Styled sections (ingredients, instructions, nutrition)
- Good typography and spacing

## 📚 What I learned
- Structuring longer pages with multiple sections
- Working with lists and content layout
- Improving typography and spacing
- Creating clean UI designs
